Dice Website

For a course lab project, I developed an interactive dice game website using HTML, CSS, and JavaScript. This project involved creating a dynamic platform where users roll two dice to achieve outcomes of win, lose, or draw based on the sum of the dice. I utilized HTML to structure the game’s layout and content, while CSS to style a visually appealing design. JavaScript was important in implementing the game’s functionality, handling dice rolls, updating the balance, and managing game logic.

The website allows users to play multiple rounds, with the ability to track and view a history of balances for each round. This balance history is visually enhanced through a color-coding mechanism: red indicates a negative balance, while green indicates a positive one. This project demonstrates my ability to effectively use front-end technologies to create a functional and interactive web application.
